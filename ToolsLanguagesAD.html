Programmiersprachen & Tools

Die Werkzeuge für die App-Entwicklung

Programmiersprachen

  • Java - stabil und weit verbreitet für Android-Apps
  • Kotlin - moderne und kompakte Alternative für Android
  • Swift - Hauptsprache für iOS-Apps
  • JavaScript - für Web- und Hybrid-Apps
  • Dart - für Flutter-Apps

Frameworks

  • Flutter - plattformübergreifend, eigene UI-Komponenten
  • React Native - JavaScript-basiert, nutzt native Komponenten
  • Ionic - Hybrid-Apps mit HTML/CSS/JS
  • Unity - vor allem für Spiele & 3D-Anwendungen

Tools

  • Android Studio - Entwicklungsumgebung für Android
  • Xcode - Entwicklungsumgebung für iOS
  • Visual Studio Code - universeller Editor
  • Git & GitHub - Versionsverwaltung & Zusammenarbeit
  • Firebase - Backend-Dienste wie Datenbanken, Anmeldung, Cloud-Funktionen
