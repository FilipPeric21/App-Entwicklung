# **Website Project Description**

## **1. Project Title**

	Minecraft-Mobs-Wiki

---

## **2. Project Idea**

	Das Minecraft-Mobs-Wiki wird eine Website, die Informationen
	über verschiedene Kreaturen aus dem Spiel "Minecraft" gibt.
	Die Website gibt Tipps gegen verschiedene Monster und zeigt,
	welche Vorteile ein Tier hat.

---

## **3. Target Audience**

	Die Zielgruppe sind Leute, die Minecraft spielen,
	vielleicht aber nicht so viel Wissen zu diesem Thema haben.
	Die Website kann Anfängern helfen, mehr über das Spiel
	herauszufinden.

---

## **4. Structure of the Website**

	Die Website wird ungefähr aus 5–8 HTML-Dateien bestehen + 1 CSS Datei

---

## **5. Design and Layout**

	Es wird eine Kopfzeile geben, in der man sich zurechtfinden kann
	und die verschiedenen Kategorien von Kreaturen anschauen kann.
	Je nach Kategorie wird sich die Hintergrundfarbe verändern.
	Die Kopfzeile wird jedoch immer oben angezeigt werden.

---

## **6. Milestones**
	
	1. Woche: Details planen  
	2. Woche: HTML-Dateien schreiben, grob die Seite fertigstellen  
	3. Woche: Mit CSS die Seite verschönern  
	4. Woche: Kleine Details ändern und die Präsentation halten
